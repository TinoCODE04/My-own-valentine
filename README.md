## How to Deploy

### Option 1: GitHub Pages (Simplest)

1. Create a new repository on GitHub
2. Push this code:
   ```bash
   git remote add origin https://github.com/YOUR_USERNAME/valentine.git
   git branch -M main
   git push -u origin main
   ```
3. Go to repository Settings â†’ Pages
4. Select "main" branch and "/ (root)" folder
5. Click Save
6. Your site will be live at: `https://YOUR_USERNAME.github.io/valentine/`

### Option 2: Netlify (Recommended)

1. Go to [netlify.com](https://netlify.com)
2. Sign up/Login (can use GitHub)
3. Click "Add new site" â†’ "Deploy manually"
4. Drag and drop the entire `Valentine-V2` folder
5. Your site will be live instantly with a random URL
6. You can customize the URL in site settings

### Option 3: Vercel

1. Install Vercel CLI: `npm install -g vercel`
2. Run: `vercel`
3. Follow the prompts
4. Your site will be deployed and you'll get a URL
